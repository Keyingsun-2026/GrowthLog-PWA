# Growth Log — 个人成长与关系档案 PWA

## 项目概述

一个面向个人使用的关系管理 + 八字分析 PWA，帮助用户记录和理解身边的人。中文界面，移动端优先（iOS PWA）。

## 文件结构

```
GrowthLog-PWA/
├── Growth Log/
│   ├── index.html      ← 整个 app，单文件，约 2800 行
│   ├── manifest.json   ← PWA manifest
│   ├── sw.js           ← Service Worker（离线缓存 + 更新提示）
│   ├── icon-192.png
│   └── icon-512.png
└── 暂不/
    └── 成长手册.html   ← 早期独立版本，暂时搁置
```

## 核心功能

**三个主 Tab：**
- **我的档案** — 设置自己的名字/生日/性别/头像，生成八字日主分析
- **我的外圈** — 管理人物档案，支持 7 种关系类型（有好感的人、爱豆、新朋友、朋友、恋人、同事、家人）
- **设置** — 账号、数据导出/导入、PIN 锁

**每个人物档案有三个子页面：**
- **日主分析** — 根据生日计算八字日主，展示性格分析；如果双方都有生日，还会显示相处配合分析
- **我感觉Ta** — 写自由文本印象，系统根据关键词 + 日主给出解读
- **相处碎片** — 记录文字/图片/情绪片段

## 技术架构

- **全部在单个 `index.html` 里**，无构建工具，无外部 JS 框架
- **后端：Supabase**（自行内嵌了轻量客户端，不依赖 CDN）
  - 三张表：`profiles`（自己）、`people`（外圈人物）、`fragments`（碎片/感受记录）
  - Session 存在 IndexedDB（比 localStorage 稳定）
  - 图片以 base64 压缩后存入 `photo_url` 字段
- **八字引擎**：纯 JS，计算年/月/日柱 + 十神，10 种日主的性格描述全部硬编码，55 对相处配合表（`CT` 对象）
- **PWA**：manifest + service worker，支持 iOS 添加到主屏幕

## 关键常量 / 全局变量

| 变量 | 说明 |
|------|------|
| `D` | 内存数据对象 `{me, people[]}`，从 Supabase 加载 |
| `curPerson` | 当前打开的人物对象 |
| `DM_DATA[0..9]` | 十天干（甲乙丙丁戊己庚辛壬癸）的性格数据 |
| `CT` | 相处配合表，key 为 `"min-max"` 格式 |
| `_SB` | 内嵌 Supabase 客户端（auth + from） |
| `_user` | 当前登录用户对象 |

## 近期修改记录

- **删除确认**：所有删除操作（碎片、感受记录、人物档案、清除所有数据）执行前均有确认弹窗

## Supabase 配置

- URL 和 anon key 硬编码在 `index.html` 顶部 `_SB_URL` / `_SB_KEY`
- 如需更换项目，替换这两个常量即可
